# Docs

Esta pasta guarda materiais de referencia que ajudam planejamento, design e revisao, mas nao fazem parte do runtime da API.

## Pastas

```text
assets/screenshots/       Capturas usadas para analise de UI e comparacao visual
design/claude-export/     Export bruto de design gerado pelo Claude
```

Os arquivos em `design/claude-export/` devem ser tratados como referencia visual. Codigo de producao deve ser implementado em `src/` ou `public/`, conforme a camada correta.
